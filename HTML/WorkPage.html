Best City Vote
Participate in the Best City Vote! A React web-app where you can tell the world what your favourite city is. This web-app uses multiple React features such as event proportion, event handling, state changes, and dealing with components throughout the component life cycle. The web-app stores all of its data using the Firebase platform.
Truth or Dare
A fun and enjoyable game of Truth or Dare, but with a twist! All the truths and dares are anonymized. Truth or Dare is an android application written in Java. This application efficiently uses multiple modules by utilizing the Model-View-Controller and Singleton design patterns.
